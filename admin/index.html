How this works — read me first

Two ways to turn a class off

Active toggle (on each class card): quietly hides it from the public site. No emails are sent and no sign-ups are removed. Use it to pause a class or hide one you’re still setting up.

Edit → Cancel series & notify: emails everyone signed up for upcoming dates, moves their sign-ups to the Archive, then turns the class off. Use this when you’re actually cancelling a class people registered for.

Recurring series vs one-off

+ New series repeats weekly. Each week’s session opens for sign-up about 7 days ahead — so a brand-new series won’t appear on the public schedule until roughly a week before its first session. That’s expected, not a bug.

+ New one-off is a single date (a workshop, a special class). It’s signable as soon as you save it, right up to the date.

Rescheduling a date

Move (on an upcoming date): reschedules just that date. If anyone is signed up, you’ll be asked whether to email them the new date/time.

Edit → Date (one-offs only): changes the date quietly — no emails. Use Move instead if people are already signed up.

Cancelling a single date

Cancel (on an upcoming date): emails the registered students (and the waitlist if you tick the box), then moves their sign-ups to the Archive. This can’t be undone from here.

Deleting a class

Edit → Delete series: permanently removes the class. It’s only allowed when no one is signed up for upcoming dates — cancel the series first if needed. Past sign-ups always stay in your Archive.

When emails go out

Sign-up confirmations and the Zoom link (30 min before online classes) are automatic. Cancel and Move only email when you confirm them here (you also get a summary). The active toggle, Edit, and Delete never email anyone.

Undoing a cancel or move

Cancels and moves are stored as “exceptions.” To reverse one (un-cancel a date, revert a move), edit the Exceptions tab in the Google Sheet.
